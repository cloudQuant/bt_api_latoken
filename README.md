# bt_api_latoken

Latoken exchange plugin for bt_api framework.

## Installation

```bash
pip install bt_api_latoken
```

## Usage

```python
from bt_api_latoken import LatokenExchangeDataSpot, LatokenRequestDataSpot
```

## Features

- REST API support
- HMAC-SHA512 authentication
- Symbol format: `BTC/USDT` -> `btc_usdt`
